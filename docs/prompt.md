記帳 Bot 專案規劃

目標

做一個可以用自然語言對話的記帳機器人（Discord Bot + LLM）

───

功能列表

1. 自動載具同步

• 串接財政部電子發票 API
• 定時抓取發票資料
• 用 LLM 解析：商家名稱 → 類別（food, shopping, transport...）+ 金額 + 日期

2. 自然語言記帳

• 用戶傳：「今天晚餐吃海底撈 500」
• Bot 解析 → 存進資料庫

3. 查帳功能

• 問：「這個月餐飲花了多少」
• 問：「比上個月多還是少」
• 問：「最近零食買太多囉」

4. 操作方式

• 指令（Slash Commands）：可以用指令控制功能
• 互動式（Buttons/Select Menus）：點擊互動介面操作
• 自然語言：用對話方式處理
• 三種方式都能用，看使用者習慣

5. 分析功能

• 消費習慣分析
• 統計圖表

初始設定（第一次使用）

Bot 會用互動式 UI + 對話方式詢問：

必填

• 每個月預算：多少錢（會釘選在頻道頂）
• 性別：用於稱呼
• 每日提醒記帳時間：幾點提醒
• 是否分不同帳本：有哪些帳本
• 類別設定：預設分類（可自訂）

預設分類

• 支出：餐飲、交通、購物、娛樂、房租/帳單、醫療、教育、投資、禮物、其他
• 收入：薪資、兼職、被動收入、紅包、禮物、生活費、其他

選填（互動式設定）

• 管理帳本：新增/刪除/編輯帳本
• 管理預算：修改每月預算
• 管理貨幣：設定貨幣單位
• 管理類別：新增/編輯/刪除分類
• 頻道名稱顯示餘額：開/關
• 每日提醒時間：修改提醒時間
• 匯入/匯出帳目：CSV/JSON 格式
• 載具同步：手動設定和開關
• 週期性收支：設定週期性收入或支出（如房租、電話費）

專用設定頻道

• 會有一個頻道放網頁連結，可以用瀏覽器直接操作帳本設定

額外功能：本地 Web Dashboard

目標

• 在本地跑一個網頁，可以瀏覽帳目數據
• 讓龍蝦可以直接查詢、分析這些資料

技術

• 後端：FastAPI / Flask (Python)
• 前端：簡單 HTML/JS 或 Streamlit
• 資料庫：SQLite

功能

• 列表顯示所有帳目
• 篩選（按日期、類別、金額範圍）
• 統計圖表
• 讓 AI 可以直接 query 資料庫

部署

• 用 Cloudflare Tunnel 暴露到外網
• 這樣在外面也能訪問

專用設定頻道

• 一個頻道專門放網頁 Dashboard 連結
• 可以用瀏覽器直接操作所有設定
